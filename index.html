Usage Guide

1. Open Product Builder → add your garment (or use the default ANA Modest Dot Set) → Save Preset.
2. Go to Generate → pick a Content Type (post, carousel, reel, grid row, ad…).
3. Select the Active Product, a Visual Theme, and a Layout.
4. For carousels / reels / grid rows, fill the builder that appears.
5. Set your Copy Overlay (headline, subheadline, CTA) and toggle the Quality rules.
6. Press ✦ Generate Prompts. Switch output tabs (Full, Short, Negative, Slides, Storyboard, Captions, Hashtags, Client Instructions).
7. Copy the prompt into your AI image/video tool. Add the ANA logo manually afterward.
8. Save to History to reuse later. Use Export to back everything up.

Remember: This dashboard creates prompts, not images. It never adds a logo — leave logo space and place it manually.
